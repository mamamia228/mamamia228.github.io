# Выбор темы 
Я выбрала тему Moonstone, потому что она показалась мне наиболее эстетичной и удобной для восприятия. Мне хотелось создать современный и аккуратный дизайн, который будет выглядеть профессионально и не отвлекать от содержания. Также я ориентировалась на удобство навигации и читаемость информации. Такой стиль оформления, на мой взгляд, лучше всего подходит для сайта IT-тематики
